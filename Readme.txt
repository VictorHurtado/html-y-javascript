Este pequeño blog  contiene algunas de las etiquetas, no incluye el plugin datatable pues no 
me parecía oportuno utilizarlo aquí. Se incluyo bootstrap para algunos botones y principalmente 
para los estilos del formulario. Contiene un css llamado estilos donde se involucran todos los 
estilos personalizados correspondientes.


Se incluyo javascript y jquery para algunas funciones, como intercambio de clases, ocultar 
información o desplegarla en su defecto, el blog contiene dos botones grandes que despliegan
 tanto mi información principal como información adicional. Por otro lado el jquery permite 
generar la animación de desplazamiento suave hacia una sección del blog. Ademas en javascript 
se añadio un botón en la sección de trabajos o fotos donde al pulsar el botón se añade un nueva
 foto con su respectiva descripción nombre e imagen, al trabajarse de manera local se utilizaron
 arreglos para almacenar los nombres, las rutas y ademas las descripciones.

al final los enlaces del footer se agregaron para redireccionar a mis redes sociales.

se incluye normalize para quitar los estilos por defecto.
